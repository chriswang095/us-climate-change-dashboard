This folder contains the data used for GHG emission analysis in the Tableau dashboard. Historic data from the US EPA Data Explorer is aggregated at four levels. Projected data from 2024 to 2035 is sourced from the US EPA via the US 2035 Nationally Determined Contribution submitted to the UNFCC.
  1. By year
  2. All-time
  3. 2024 only
  4. Projection
